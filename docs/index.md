# Dataspace Ecosystem KIT

Welcome to the **Dataspace Ecosystem KIT** (Keep It Together) documentation. This comprehensive guide provides everything you need to understand, deploy, and operate the Dataspace Ecosystem.

## What is Dataspace Ecosystem?

The Dataspace Ecosystem is a comprehensive solution for building and managing sovereign data spaces, enabling secure and controlled data exchange between organizations while maintaining data sovereignty and trust.

## Key Components

| Component | Description |
|-----------|-------------|
| **Control Plane** | Manages contracts, policies, and assets for data exchange |
| **Data Plane** | Handles actual data transfer between participants |
| **Federated Catalog** | Enables discovery of data offerings across the dataspace |
| **Identity Hub** | Manages decentralized identities and credentials |
| **Issuer Service** | Issues and verifies verifiable credentials |
| **Telemetry Service** | Collects and processes telemetry data |
